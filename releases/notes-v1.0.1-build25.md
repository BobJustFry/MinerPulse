## Miner Pulse 1.0.1 (25)

Desktop monitor for WhatsMiner, Antminer, and Avalon ASIC miners.

### Highlights

- Real-time dashboard: hashrate, temperatures, fans, shares, boards
- Chip matrix with temperature, voltage, and solutions views
- Live charts with poll/record/session playback (Service tier)
- Network scan, log import (.mpulse), drag & drop
- Custom title bar, light/dark themes, RU / EN / 中文
- Signed auto-updater (manifest updated separately when signature is available)
- About dialog with update check, GitHub, Telegram, and donation

### Install

Download **MinerPulse_1.0.1_x64-setup.exe** below (Windows x64, NSIS).
