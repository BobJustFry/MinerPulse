## Miner Pulse 1.0.1 (59)

Desktop monitor for WhatsMiner, Antminer, and Avalon ASIC miners.

### Highlights

- **Subscription auth in desktop app** — click the tier badge (FREE) to open login: activation code from [mpulse.bob4.fun](https://mpulse.bob4.fun) or email + password
- **WhatsMiner chip status analysis** — PLL/CRC/repeat/temp badges on the chip matrix with legend (RU / EN / 中文)
- Platform: web registration, admin CRUD, license API (build 58)
- Real-time dashboard, chip matrix, charts, network scan, log import, auto-updater

### Install

Download **MinerPulse_1.0.59_x64-setup.exe** below (Windows x64, NSIS). Users on build 45 or older can update from **About → Check for updates**.
