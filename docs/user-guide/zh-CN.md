# Miner Pulse 用户指南

**Miner Pulse** 是 Windows 桌面应用，用于通过网络监控 ASIC 矿机（WhatsMiner、Antminer、Avalon 及其他 cgminer 兼容设备）。

版本号和构建号显示在窗口标题：`Miner Pulse X.Y.Z (BBB)`。

## 安装

1. 打开 [Releases](https://github.com/BobJustFry/MinerPulse/releases/latest)。
2. 下载 **`MinerPulse_*_x64-setup.exe`**。
3. 运行安装程序（NSIS）。
4. 启动 **Miner Pulse**。

> 首次运行时 Windows 可能请求防火墙权限 — 请允许访问本地网络，否则轮询和扫描无法工作。

---

## 快速开始

1. 输入矿机 **IP** 和 **端口**（默认 `4028`）。
2. 点击 **读取** 进行一次性遥测读取。
3. 数据出现在 **数据** 标签页；有芯片数据时显示 **芯片** 标签页。

持续监控请使用 **轮询** 模式。WhatsMiner 首次连接可能需要 API 设置 — 见 [WhatsMiner](#whatsminer-首次连接)。

## 工具栏

| 控件 | 说明 |
|------|------|
| **IP : 端口** | 矿机地址。cgminer API 端口通常为 `4028`。 |
| **搜索** | 扫描子网并从列表选择矿机。 |
| **读取** | 单次读取，无后台轮询。 |
| **轮询** | 周期性轮询，频率 1–15 次/秒。 |
| **录制** | 轮询并保存会话为 `.mprs`。 |
| **停止** | 停止轮询或录制。 |
| **打开** | 打开录制、快照、日志或旧版 `.mpulse`（自动识别）。 |
| **保存** | 将当前快照保存为 `.mpsn`。 |
| **主题** | 浅色 / 深色。 |
| **紧凑** | 紧凑工具栏：图标 + 悬停提示。 |
| **语言** | RU / EN / 中文。 |
| **订阅** | 管理套餐（tier 徽章）。 |
| **关于** | 版本、更新、捐赠、诊断。 |

### 操作模式（分体按钮 ▾）

- **读取** — 单次。
- **轮询** — 实时图表与数据刷新。
- **录制** — 轮询并写入文件（需 Client 或 Service 套餐）。

选择 **轮询** 或 **录制** 时可设置 **频率**（1/3/5/10/15 次/秒）。

## 标签页

### 数据

概览：型号、固件、状态、算力、温度、风扇、功耗、算力板、矿池、扩展参数（电源、模式、频率等）。

### 芯片

按板的芯片矩阵（温度、电压、错误）。无芯片数据时隐藏此标签。

### 控制台

原始日志与 API 响应，便于诊断。

### 图表

算力、板温、功耗、风扇转速。轮询/录制时实时更新；回放时可拖动时间轴。

布局：**平铺** 或 **列表**。

### 命令

矿机控制（重启、频率、工作模式等）— 仅 **Avalon**，需 Client/Service 套餐。

## 录制与文件

| 扩展名 | 类型 | 说明 |
|--------|------|------|
| `.mprs` | 会话录制 | 完整轮询历史：全部字段、芯片、控制台。 |
| `.mpsn` | 快照 | 单个时间点。 |
| `.mpulse` | 旧版 | JSON 格式，仍支持打开。 |
| `.txt` / `.log` | 日志 | 导入 Avalon/Antminer 日志。 |

### 回放

1. **打开** → 选择 `.mprs`。
2. 自动打开 **图表** 标签，底部显示回放栏。
3. **播放** / **暂停**、拖动、速度 0.5×–8×。
4. **数据**、**芯片**、**控制台** 显示当前帧状态。

### 拖放

可将文本日志（`.txt`、`.log`）拖入窗口。`.mprs` 请用 **打开** 按钮。

## 网络扫描

1. 点击主界面 **搜索**。
2. 选择子网或 **自定义 IP 范围**。
3. **★** 将范围保存到 **收藏**。
4. **搜索** 开始扫描；双击行可将 IP 填入主窗口。

列表中完全支持：Avalon、Antminer、WhatsMiner。其他 cgminer 设备可能显示但功能有限。

## 支持的矿机

| 系列 | 完整读取 | 芯片 | 命令 |
|------|:--------:|:----:|:----:|
| **Avalon** (嘉楠) | ✅ | ✅ | ✅ |
| **Antminer** (比特大陆) | ✅ | ✅* | — |
| **WhatsMiner** (微比特) | ✅ | ✅** | — |
| **其他 cgminer** (LuxOS、Braiins 等) | ⚠️ 基础 | — | — |
| **Innosilicon** | ❌ | — | — |

\* 取决于型号与固件。  
\** WhatsMiner 芯片可能需要 LuCI/API 配置。

**自定义固件：** 若在 4028 端口使用 cgminer 协议但未识别为 Antminer/Avalon/WhatsMiner，则仅 **基础** 数据（算力、矿池、温度），无芯片矩阵与命令。

## WhatsMiner 首次连接

1. 点击 **读取** 或 **轮询**。
2. 如需设置，会显示 **WhatsMiner 设置** 对话框：
   - LuCI 用户名/密码（默认 `admin` / `admin`）；
   - **测试登录**；
   - **保存并读取**。
3. 登录账户后可同步凭据到云端。

无 LuCI/API 时芯片可能为空，但算力和矿池通常仍可通过 TCP API 获取。

## 技巧与故障排除

| 问题 | 建议 |
|------|------|
| 无响应 | 检查网络、ping、4028 端口、Windows 防火墙。 |
| WhatsMiner 长时间「读取中」 | 等待最多 15 秒或 **取消** 后重试；在浏览器测试 LuCI。 |
| 无芯片数据 | WhatsMiner：配置 API；Antminer/Avalon：固件可能不返回 devs。 |
| 无法打开录制 | 使用 **打开**（`.mprs` 不支持拖放）。需 Client+ 套餐。 |
| 订阅错误 | 检查网络与网站设备数量限制。 |

**诊断日志：** 关于 → 发送日志（需登录）。在网站个人中心查看。
