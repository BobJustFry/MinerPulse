# Руководство пользователя Miner Pulse

**Miner Pulse** — настольное приложение для Windows: мониторинг ASIC-майнеров по сети (WhatsMiner, Antminer, Avalon и другие cgminer-совместимые устройства).

Версия и номер сборки отображаются в заголовке окна: `Miner Pulse X.Y.Z (BBB)`.

## Установка

1. Откройте [страницу Releases](https://github.com/BobJustFry/MinerPulse/releases/latest).
2. Скачайте **`MinerPulse_*_x64-setup.exe`**.
3. Установите программу (установщик NSIS).
4. Запустите **Miner Pulse**.

> При первом запуске Windows может запросить разрешение брандмауэра — разрешите доступ к локальной сети, иначе опрос и поиск не сработают.

---

## Быстрый старт

1. Введите **IP** майнера и **порт** (по умолчанию `4028`).
2. Нажмите **Читать** — однократное чтение телеметрии.
3. Данные появятся на вкладке **Данные**; при наличии чипов откроется вкладка **Чипы**.

Для непрерывного мониторинга используйте режим **Опрос** (см. ниже). Для WhatsMiner при первом подключении может понадобиться окно настройки API — см. [WhatsMiner](#whatsminer-первый-запуск).

## Панель инструментов

| Элемент | Описание |
|---------|----------|
| **IP : Порт** | Адрес майнера. Порт cgminer API обычно `4028`. |
| **Поиск** | Сканирование подсети и выбор майнера из списка. |
| **Читать** | Одно чтение без фонового опроса. |
| **Опрос** | Периодический опрос с выбранной частотой (1–15 раз/с). |
| **Запись** | Опрос + сохранение сессии в файл `.mprs`. |
| **Стоп** | Остановить опрос или запись. |
| **Открыть** | Открыть запись, снимок, лог или старый `.mpulse` (тип определяется автоматически). |
| **Сохранить** | Сохранить текущий снимок в `.mpsn`. |
| **Тема** | Светлая / тёмная. |
| **Компакт** | Компактный тулбар: иконки вместо подписей; подсказка при наведении. |
| **Язык** | RU / EN / 中文. |
| **Подписка** | Управление тарифом (значок tier). |
| **О программе** | Версия, обновления, донат, диагностика. |

### Режимы действия (кнопка с меню ▾)

- **Читать** — разовое чтение.
- **Опрос** — живые графики и обновление данных.
- **Запись** — опрос с записью в файл (нужен тариф Client или Service).

При **опросе** и **записи** доступен выбор **частоты** (1/3/5/10/15 в секунду).

## Вкладки

### Данные

Сводка: модель, прошивка, статус, хешрейт, температуры, вентиляторы, мощность, платы, пулы, расширенные параметры (PSU, режим, частота и т.д.).

### Чипы

Матрица чипов по платам (температура, напряжение, ошибки). Вкладка скрыта, если у майнера нет данных о чипах.

### Консоль

Сырой лог и ответы API — удобно для диагностики.

### Графики

Хешрейт, температуры плат, мощность, вентиляторы. При опросе и записи значения обновляются в реальном времени; при воспроизведении записи — курсор по таймлайну.

Раскладка: **плитка** или **список** (кнопки на панели графиков).

### Команды

Управление майнером (перезагрузка, частота, режим и т.д.) — только для **Avalon** и тарифа **Client/Service**.

## Записи и файлы

| Расширение | Тип | Описание |
|------------|-----|----------|
| `.mprs` | Запись сессии | Полная история опроса: все поля, чипы, консоль. |
| `.mpsn` | Кадр (снимок) | Один момент времени. |
| `.mpulse` | Устаревший | Старый JSON-формат; по-прежнему открывается. |
| `.txt` / `.log` | Лог | Импорт лога Avalon/Antminer. |

### Воспроизведение записи

1. **Открыть** → выберите `.mprs`.
2. Откроются **графики** и панель воспроизведения внизу.
3. **Воспр.** / **Пауза**, перемотка, скорость 0.5×–8×.
4. На вкладках **Данные**, **Чипы**, **Консоль** — состояние выбранного кадра.

### Перетаскивание файлов

Можно перетащить текстовый лог (`.txt`, `.log`) в окно приложения. Бинарные `.mprs` пока открывайте через кнопку **Открыть**.

## Поиск в сети

1. Нажмите **Поиск** на главной панели.
2. Выберите подсеть или **свой диапазон IP**.
3. Кнопка **★** — сохранить диапазон в **Избранное**.
4. **Поиск** — сканирование; найденные майнеры в таблице.
5. Двойной щелчок или кнопка подключения — IP подставится в главное окно.

Поддерживаемые в списке: Avalon, Antminer, WhatsMiner. Другие cgminer-устройства могут отображаться, но с ограниченной поддержкой.

---

## Поддерживаемые майнеры

| Семейство | Полное чтение | Чипы | Команды |
|-----------|:-------------:|:----:|:-------:|
| **Avalon** (Canaan) | ✅ | ✅ | ✅ |
| **Antminer** (Bitmain) | ✅ | ✅* | — |
| **WhatsMiner** (MicroBT) | ✅ | ✅** | — |
| **Другие cgminer** (LuxOS, Braiins и т.д.) | ⚠️ базово | — | — |
| **Innosilicon** | ❌ | — | — |

\* Зависит от модели и прошивки.  
\** Для чипов WhatsMiner может потребоваться настройка LuCI/API.

**Кастомные прошивки:** если майнер отвечает на порту 4028 в формате cgminer, но не распознан как Antminer/Avalon/WhatsMiner — читаются **базовые** данные (хешрейт, пулы, температуры). Без матрицы чипов и команд.

Подробная таблица для разработчиков: см. обсуждение в Issues или `minerpulse-core` drivers.

## WhatsMiner: первый запуск

1. Нажмите **Читать** или **Опрос**.
2. Если нужна настройка — появится окно **Настройка WhatsMiner**:
   - логин/пароль LuCI (по умолчанию `admin` / `admin`);
   - **Проверить вход** — тест учётных данных;
   - **Сохранить и читать** — сохранение и повторное чтение.
3. Учётные данные можно синхронизировать с облаком (при входе в аккаунт).

Без доступа к LuCI/API чипы могут быть пустыми, но хешрейт и пулы обычно доступны через TCP API.

## Подписка

| Тариф | Опрос / запись | Графики | Воспроизведение | Команды |
|-------|:--------------:|:-------:|:---------------:|:-------:|
| **Free** | Разовое чтение (интервал ≥10 с) | — | — | — |
| **Client** | Опрос + запись `.mprs` | ✅ | ✅ | ✅ (Avalon) |
| **Service** | То же | ✅ | ✅ | ✅ |

Активация: значок tier → **Управление подпиской** → код с [mpulse.bob4.fun](https://mpulse.bob4.fun) или вход по email.

## Обновления

**О программе** → **Проверить обновления**.

Обновления подписаны криптографически и загружаются с GitHub Releases. Манифест: `releases/update.json` в репозитории.

---

## Советы и решение проблем

| Проблема | Что попробовать |
|----------|-----------------|
| Нет ответа по IP | Проверьте сеть, пинг, порт 4028, брандмауэр Windows. |
| Долгое «Чтение…» (WhatsMiner) | Подождите до 15 с или **Отмена** и повтор; проверьте LuCI в браузере. |
| Нет чипов | WhatsMiner: настройка API; Antminer/Avalon: модель/прошивка может не отдавать devs. |
| Запись не открывается | Используйте **Открыть** (не drag-drop для `.mprs`). Нужен тариф Client+. |
| Ошибка подписки | Проверьте интернет и лимит устройств на сайте. |

**Диагностический лог:** О программе → отправка лога (нужен вход в аккаунт). Просмотр в личном кабинете на сайте.
