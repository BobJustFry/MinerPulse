## Про кнопку «Fork» на GitHub

Публичный репозиторий технически можно форкнуть кнопкой GitHub — **это ограничение платформы**, а не разрешение.

**Юридически:** форк, копирование, перепубликация и производные продукты **запрещены** без письменного согласия правообладателя (Bobrov Andrey / BobJustFry).

Наличие форка **не даёт права**:
- использовать код в своём продукте;
- обходить подписку или лицензирование;
- распространять сборки под другим именем.

## Официальное распространение

Единственный авторизованный канал для пользователей:

1. [GitHub Releases](https://github.com/BobJustFry/MinerPulse/releases) — подписанные `.exe` / `.msi`
2. Авто-обновление через подписанный [`releases/update.json`](releases/update.json)

Любые другие сборки, зеркала и «репаки» **не являются официальными**.
